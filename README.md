# ChoubaWealth 📈

**ChoubaWealth** my partner is bad at money management so i made this to simplify tracking his stocks and savings, thx

---

## Key Features

*   **Stock Portfolio Tracker:** Monitor your holdings across international markets with the most updated price data.
*   **Savings Management:** Keep a consolidated view of your cash reserves and liquid assets.
*   **Live USD to IDR Conversion:** Automatically converts foreign asset values into Rupiah using real-time exchange rates.
*   **Consolidated Dashboard:** View your total net worth across different asset classes with a single command.

## Tech Stack

*   **Language:** Python / Node.js
*   **Library:** Discord.py / Discord.js
*   **APIs:** Market Data (e.g., Yahoo Finance) & Exchange Rate APIs
